# Active Areas

Which areas are currently being modified by agents. **Not a lock** — just visibility to reduce conflicts when running parallel agents.

## How to use

**At session start:** Read this file. If another agent is working on files you plan to touch, flag it to the user.

**When you start modifying files:** Add your area below with a timestamp and brief scope description.

**At session end:** Remove your entry (or leave it — the user cleans this periodically).
